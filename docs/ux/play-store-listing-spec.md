## Asset Rules

1. Only show validated features from WP-11/WP-12 evidence.
2. No broad voice/STT/TTS claims in the public MVP listing; any limited-beta voice mention stays in closed-track notes only.
3. No cloud-sync claims.
4. No asset should imply richer direct-tool dispatch or broader image-analysis scope than the validated prompt-first/single-image launch surface.
5. Benchmark claims require current dated Stage-2 closure evidence links (do not use stale numbers).
